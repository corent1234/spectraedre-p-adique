\section{Questions from the reviews}

\begin{itemize}
\item
  Lemma 3.5 (DIproj): The proof uses the term ``coarsest'' (or ``coarse'') to describe the
ball that results from intersecting several valuation inequalities. Since this is a technical
notion specific to ultrametric geometry, it should be explicitly defined. The proof relies
on the ultrametric property that the intersection of a nested family of balls is the smallest
ball; the wording should be adjusted accordingly to avoid confusion.
\item
  Theorem 4.6: I recommend to add a brief discussion of termination and complexity.
  \simone{Discussion on termination has been added in the camera ready version}. 
\item
  Theorem 1.2: The result assumes the residue field $\kappa$ is infinite. It would be helpful to
  add a remark or a small example discussing what happens when $\kappa$ is finite
  (e.g., $\kappa = \mathbb{F}_p$). Does the statement still hold, or are there counterexamples?
  A short comment would guide readers who work with fields like $\mathbb{Q}_p$.
\item
  Introduction (motivation): The paper would be stronger if the introduction provided
more concrete motivation. For instance, what is the geometric or algorithmic significance
of inequalities of the form $val(\ell(x)) \geq 0$? Are there applications in non-Archimedean
geometry, p-adic optimization, or coding theory? Or is the work primarily a formal
analogue of real linear/semidefinite programming? Clarifying the ``why'' would help the
reader appreciate the results.
\end{itemize}

\section{Size/precision in LP}

\subsection{Real case}
This material is from \cite[Ch.4]{korte2008combinatorial}.
Define $\size(n) = 1+\lceil\log(|n|+1)\rceil$ for $n \in \ZZ$ and
$\size(r) = \size(p)+\size(q)$ for $r=\frac{p}q \in \QQ$ with $p,q \in \ZZ$ coprime.
For a vector $x = (x_1,\ldots,x_n) \in \QQ^n$ define $\size(x) = n+\sum_i \size(x_i)$
and for a matrix $A \in \QQ^{m \times n}$, define $\size(A) = mn+\sum_{ij} \size(a_{ij})$.

Consider a dual LP in standard form:
\begin{equation}\label{dualLP}
d^* = \sup b^Ty, \,\,\,\, A^Ty \leq c
\end{equation}
for some matrix $A \in \QQ^{m \times n}$ and vectors $b \in \QQ^m$, $c \in \QQ^n$.

\begin{theorem}[{\cite[Thm.4.4]{korte2008combinatorial}}]
  Assume that the linear program in \eqref{dualLP} has an optimal solution.
  Then it has an optimal solution $x$ such that $\size(x) \leq 4n(\size(A)+\size(c))$
  with components of size at most $4(\size(A)+\size(c))$.
\end{theorem}

Note that the feasibility problem asking whether problem \eqref{dualLP} is feasible
has the same size estimates (take $b=0$).
